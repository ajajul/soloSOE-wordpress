== Description ==

Profile Builder - Import Export Add-On

This plugin adds a subpage where you can Import and Export settings of Profile Builder.


== Installation ==

1. Upload pb-add-on-import-export folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Profile Builder => Import and Export subpage and start importing or exporting your options.
