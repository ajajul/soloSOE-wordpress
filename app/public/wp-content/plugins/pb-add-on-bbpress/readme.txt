== Description ==

Profile Builder - bbPress Integration Add-on

Integrates Profile Builder with bbPress, allowing you to transfer bbPress user profile to Profile Builder user profile automatically.

All forum user links are automatically redirected to the extended user profiles created with Profile Builder.
Also, you can link the "Edit" user page from bbPress to the "Edit Profile" form created with Profile Builder.


NOTE:

In order to function properly this plugin requires that you have both bbPress and Profile Builder plugins installed

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to bbPress tab under Profile Builder menu and you'll have access to bbPress integration settings.
